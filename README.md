# Macros in BNF
Yep. No introduction is needed as it pretty much does what it says on the tin.

## Installation
1. Open the **extensions panel**.
2. Click **Install extension**.
3. Paste in the repo's URL:
```
https://github.com/CasualAutopsy/SillyTavern-Macros-in-BNF
```
4. Click **Install just for me**.
5. Done! You can now use macros in grammar strings!
